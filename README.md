# Nexcart - E-commerce Website

## Description
Nexcart is a modern full-stack ecommerce web application that includes AI-powered virtual try-on, allowing users to preview outfits before purchasing.

## Features
- User Authentication (Login / Signup)
- Add to Cart System
- Product Detail Page
- AI Virtual Try-On
- Order Management System
- Invoice PDF Download
- User Profile Dashboard

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js, Express  
Database: MongoDB  
AI: Stable Diffusion  

## Installation

1. Clone the repository
2. Run:
   npm install
3. Create a .env file
4. Start server:
   npm start
5. Open project in browser

## API Endpoints
- /api/auth/login
- /api/auth/register
- /api/cart/add
- /api/orders/create
- /api/tryon/generate
