## Data Flow

1. User interacts with frontend `id.html` to select a product and upload a photo.
2. Frontend sends a multi-part form request to `[POST] /api/try-on` in `routers/api.py`.
3. The API passes the request to `ai_service.py`.
4. `ai_service.py` forwards the image parameters to the Hugging Face IDM-VTON space.
5. The API polls for results, and the resulting image URL is passed back to the frontend.
